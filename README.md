# E-commerce Dashboard Project

A complete end-to-end project using MongoDB, Flask REST API, Streamlit UI, and a Jupyter Notebook for analytics.

This project demonstrates how to build a simple but realistic e-commerce system where:

-Data is stored in MongoDB (Users, Products, Carts)

-Flask acts as a backend API exposing CRUD + aggregation results

-Streamlit builds an interactive dashboard

-Jupyter Notebook explores the data, performs aggregations, and tests the API

### Installation

git clone <repo-url>

cd ecommerce

python -m venv venv

.\venv\Scripts\activate

### Run Flask API

python app.py

### Run Streamlit

streamlit run streamlit_app.py
